# Start Here

## Step 1: Install Dependencies

```bash
pip install -r requirements.txt
```

Or use the install script:
- **Windows**: double-click `install.bat`
- **Linux/Mac**: `bash install.sh`

## Step 2: Run the Dashboard

```bash
streamlit run app.py
```

A browser window opens automatically at `http://localhost:8501`.

## Step 3: Explore

- The app runs in **Demo Mode** by default with simulated options data
- Use the sidebar to switch symbols (SPX, SPY, QQQ, IWM)
- Adjust thresholds with the sliders
- Toggle auto-refresh on/off
